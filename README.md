# -OmniWhale
An enterprise-grade, lightweight on-chain analytics bot that monitors Ethereum, BNB Chain, and Solana in real-time. OmniWhale detects massive liquidity movements, native coin transfers, and token swaps (ERC-20, BEP-20, SPL) exceeding user-defined thresholds. Bycross-referencing transactions against an internal database of known exchange hot wallets
